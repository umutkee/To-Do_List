# To-Do List (Task-Manager) | Yapılacaklar Listesi
Bu proje, görevlerinizi yönetebileceğiniz, önceliklendirebileceğiniz ve takip edebileceğiniz modern, kullanıcı dostu bir "Yapılacaklar" (To-Do) uygulamasıdır.

Özellikler
* Görev Yönetimi: Görev ekleme, düzenleme ve silme.

* Kategorizasyon: İş, kişisel, alışveriş, sağlık gibi farklı kategorilerle düzenleme.

* Önceliklendirme: Yüksek, normal ve düşük öncelikli görev tanımlama.

* Filtreleme ve Arama: Görevleri durumuna (tümü, aktif, tamamlanan, bugün, gecikmiş) göre filtreleme ve arama yapma.

* İlerleme Takibi: Tamamlanan görevleri görsel bir ilerleme çubuğu ile görme.

* Karanlık Mod: Kullanıcı tercihine göre tema değiştirme desteği.

* Veri Kalıcılığı: Görev verileri sunucu taraflı JSON dosyasında (tasks.json) saklanır.

* Klavye Kısayolu: Ctrl + / tuş kombinasyonu ile hızlıca arama kutusuna odaklanma.

Teknolojiler
* Frontend: HTML5, CSS3, JavaScript (Vanilla)

* Backend: PHP

Kurulum
* Bu projeyi sunucunuza (PHP destekli bir ortama) kopyalayın.

* tasks.json dosyası otomatik oluşturulacaktır. (Sunucuda yazılabilir olduğundan emin olun.)

* Projeyi bir tarayıcı üzerinden açarak kullanmaya başlayın.
